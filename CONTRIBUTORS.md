# Contributors

Neutrx is a security-first Node.js 22+ HTTP client project. This file recognizes project ownership and accepted contributions.

## Project Owner

- Xenial-Devil, repository owner and final authority for project direction, releases, package publishing, license permissions, branding, and governance.

## Contributors

Contributors may be recognized for accepted contributions such as:

- Code changes.
- Tests.
- Documentation.
- Security reports handled through the private process.
- Issue triage.
- Reproduction cases.
- Design review.

Add contributor names or handles after contributions are accepted by maintainers.

## Recognition Rules

- Recognition is based on meaningful accepted contributions.
- Security reporters may be credited only with their permission.
- Maintainers may correct, remove, or reorder entries when needed.
- Automated, spam, duplicate, or rejected submissions do not require credit.
- Contribution size does not determine governance authority.
